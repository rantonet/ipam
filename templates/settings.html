Server DNS

Usati per il reverse lookup PTR durante la scansione. Se il primario risponde NXDOMAIN, viene interrogato il secondario.

Probe TCP — Bypass Firewall

Se un host blocca il ping ICMP (regola firewall), lo scanner tenta una connessione TCP sulle porte elencate. Un host che risponde su almeno una porta viene comunque marcato come attivo, evitando falsi "disponibili".

Porte tipiche: 22 SSH · 80 HTTP · 443 HTTPS · 445 SMB · 3389 RDP · 135 RPC · 8080/8443 HTTP alt

Scansione Globale
